In JavaScript, 
the Control flow is a way of how your computer runs code from top to bottom. 
It starts from the first line and ends at the last line 
unless it hits any statement that changes the control flow of the program 
such as loops, conditionals, etc.

We can control the flow of the program through any of these control structures:

1. Sequential (default mode)
2. Conditional Statements
3. Exception Handling
4. Loops and Iterations
