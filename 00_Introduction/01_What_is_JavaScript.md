### Key Characteristics
- High-Level: It abstracts away most of the complex machine details (like memory management) so you can focus on logic.
* Just-In-Time (JIT) Compiled: While historically called an "interpreted" language, modern engines (like V8) compile JS into machine code at runtime for massive performance gains.
* Multi-paradigm: It supports event-driven, functional, and object-oriented programming styles.
* Single-Threaded but Asynchronous: It uses an Event Loop to handle multiple tasks (like API calls) without blocking the main execution thread.
