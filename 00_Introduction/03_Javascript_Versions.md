1. The Historical Landmarks
ES3 (1999): The "Old World." Introduced try-catch, switch, and basic regex. Most "Legacy JS" follows this.
ES5 (2009): The "Standard World." Added strict mode, JSON support, and basic array methods (forEach, map, filter). No transpilation was needed for a decade.
ES6 / ES2015: The Big Bang. This was the largest update in history. It gave us const/let, Arrow Functions, Classes, Promises, Modules (import/export), and Template Literals.
