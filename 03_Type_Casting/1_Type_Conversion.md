### Type Conversion (Explicit) (Explicit Type Casting)
This is when you intentionally 
change the data type using built-in functions. 
It is predictable and readable.

String   to Number  : Number("42") or parseInt("42px").
Number   to String  : String(123) or (123).toString().
Boolean  to String  : String(true) becomes "true".
The Unary Plus Trick: + "50" results in 50.
